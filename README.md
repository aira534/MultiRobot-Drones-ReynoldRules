# MultiRobot_Drones
**Academic Project | Multi-Robot Systems Course at University of Zagreb (January 2025)**

This repository contains the implementation of `Reynolds' Flocking Rules` (Cohesion, Separation, and Alignment) for the distributed coordination of multiple Crazyflie drones using `ROS 2 Humble`. It also includes a navigation feature and an obstacle avoidance rule.

This is the the code used for flying the real Crazyflies drones, experiment that was conducted at **Ericsson Nikola Tesla Laboratory** in Zagreb, Croatia.

## Environment
For the development of this project, a docker was provided by the university which contained the implementation of the simulator and the Crazyflies packages in ROS2 Humble.
